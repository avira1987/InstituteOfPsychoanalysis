                    شرح کار انجام شده - پروژه انستیتو روانکاوری تهران
                              گزارش وضعیت پروژه (فوریه ۲۰۲۵)
═══════════════════════════════════════════════════════════════════════════════

■ خلاصه پروژه
─────────────────────────────────────────────────────────────────────────────
سیستم اتوماسیون آموزشی مبتنی بر متادیتا (Meta-Driven) برای انستیتو روانکاوری تهران
که روی ۷ فرایند SOP (دستورالعمل استاندارد) متمرکز است.

■ معماری برنامه‌ریزی‌شده
─────────────────────────────────────────────────────────────────────────────
• Backend:        FastAPI + SQLAlchemy (async) + PostgreSQL
• Core Engine:     Metadata-driven State Machine با Rule Engine
• Frontend:        React + Vite (پنل ادمین)
• Infrastructure:  Docker Compose + Redis + Alembic migrations

■ فرایندهای SOP (۷ عدد)
─────────────────────────────────────────────────────────────────────────────
۱. مرخصی آموزشی موقت از ثبت‌نام در کلاس‌ها          (educational_leave)
۲. آغاز درمان آموزشی                               (start_therapy)
۳. مدیریت تغییرات درمان آموزشی                     (therapy_changes)
۴. برگزاری جلسه اضافی                              (extra_session)
۵. پرداخت جلسات آتی                                (session_payment)
۶. حضور و غیاب                                     (attendance_tracking)
۷. تعیین تکلیف هزینه جلسه                          (fee_determination)

═══════════════════════════════════════════════════════════════════════════════
                        نقص‌ها و موارد ناتمام
═══════════════════════════════════════════════════════════════════════════════

• Alembic Migrations
  فایل‌های migration در alembic/versions/ دیده نشد؛ احتمالاً فقط env.py
  و script.py.mako موجود است.

• Celery
  در requirements.txt موجود است ولی استفاده مستقیمی در کد دیده نشد
  (background jobs برای SLA monitoring یا notifications).

• اتصال واقعی SMS/Email
  notification_service فقط placeholder دارد و APIهای خارجی متصل نشده‌اند.

• Admin UI Build
  در صورت نبود پوشه admin-ui/dist، سرور FastAPI فقط API را سرو می‌کند.
  برای سرو شدن UI باید در admin-ui/ دستور npm run build اجرا شود.

• فایل .env.example
  در README ذکر شده؛ وجود آن تأیید نشد.
