# فرایندها (Processes)

هر فایل JSON در این پوشه یک فرایند را تعریف می‌کند. اسکریپت seed این فایل‌ها را می‌خواند و در دیتابیس ذخیره می‌کند.

## همگام‌سازی با قوانین

هر **condition** که در آرایهٔ `conditions` ترانزیشن‌ها استفاده می‌شود، باید **حتماً** در فایل `metadata/rules/all_rules.json` با همان مقدار در فیلد `code` تعریف شده باشد.

- قبل از اضافه کردن فرایند جدیدی که در ترانزیشن‌ها از `conditions` استفاده می‌کند، قوانین مربوط را در `all_rules.json` اضافه کنید.
- در غیر این صورت اجرای seed با خطا متوقف می‌شود تا از باگ منطقی (ترانزیشن با قانون تعریف‌نشده) جلوگیری شود.
