# یادداشت فلوچارت

## Swimlaneها
1. معاون مدیر آموزش
2. مسئول سایت
3. درمانگر آموزشی
4. LMS

## پیش‌بررسی‌ها (LMS)
- وقفه؟ → بسته
- کنسلی؟ → بسته
- پرداخت نشده؟ → غیبت خودکار + fee_determination
- پرداخت شده؟ → درمانگر

## تصمیم درمانگر
- ثبت کرد؟ بله → حاضر یا غایب
- ثبت نکرد؟ → مسئول سایت

## اسکیلیشن
- مسئول سایت < ۲ روز پیگیری → بازگشت به درمانگر
- مسئول سایت ≥ ۲ روز بدون پیگیری → معاون مدیر آموزش
