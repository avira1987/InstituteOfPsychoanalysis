# یادداشت فلوچارت

## Swimlaneها
1. دانشجو
2. مسئول سایت
3. LMS
4. کمیته درمان آموزشی و سوپرویژن

## Trigger
غیبت بدون کنسل کردن جلسه

## LMS
- وقفه؟ → متوقف
- کمتر از ۲ پیوسته؟ → SMS به دانشجو، پایان
- ۲+ پیوسته؟ → پیام به مسئول سایت

## مسئول سایت
تعیین وضعیت و انتخاب ۱ یا ۲ یا ۳:
- **۱:** SMS دانشجو، violation
- **۲:** آزادسازی، پیام کمیته
- **۳:** آزادسازی، تایمر ۳ هفته، بازگشت؟ خیر → پیام کمیته

## کمیته
واگذار کردم → مجری → الف (قطع) یا ب (بازگشت) → هر دو violation
