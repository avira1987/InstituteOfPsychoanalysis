# فلوچارت: تکمیل دوره‌های ۵۰ ساعته سوپرویژن فردی

## Swimlaneها
- **دانشجو:** پرداخت پیش از جلسه
- **سوپروایزر:** ثبت حضور/غیاب، تکمیل فرم ارزیابی
- **مسئول سایت:** پیگیری عدم ثبت
- **LMS:** قفل/باز کردن، ارزیابی، اسکیت

## گره‌های تصمیم
1. وقفه یا کنسلی؟ → recording_closed
2. پرداخت شده؟ خیر → auto_absence_unpaid
3. جلسه ۴۸؟ → session_completed + unlock 50th
4. جلسه ۴۹؟ → evaluation_pending
5. عدم ثبت؟ → site_manager → deputy
