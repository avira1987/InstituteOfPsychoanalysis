# یادداشت فلوچارت

## Swimlaneها
1. دانشجو (کنسل جلسه)
2. درمانگر آموزشی (ثبت غیبت)
3. LMS

## نقاط ورود
- کنسل کردن جلسه درمان آموزشی (دانشجو)
- ثبت غیبت در جلسه درمان آموزشی (درمانگر)

## تصمیم ۱: پرداخت کرده؟
- بله → تصمیم ۲
- خیر → تصمیم ۲ (همان سوال)

## تصمیم ۲: در بازه غیبت/کنسلی مجاز؟
- بله + پرداخت کرده → یک جلسه بستانکار (سناریو ۱)
- بله + پرداخت نکرده → بدون اقدام مالی (سناریو ۲)
- خیر + پرداخت کرده → هزینه سوخت (سناریو ۳)
- خیر + پرداخت نکرده → بدهکار / کسر از بستانکاری (سناریو ۴)

## ثبت نهایی
همه مسیرها → +۱ به شمار غیبت/کنسلی (فیلد واحد).
