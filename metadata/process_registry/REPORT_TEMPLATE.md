# قالب گزارش فرایندها

برای دریافت گزارش، بگویید: **«گزارش فرایندها را بده»**

---

## محتوای گزارش (خروجی مورد انتظار)

### ۱. خلاصه آماری
| مورد | تعداد |
|------|-------|
| فرایندهای ثبت‌شده در registry | |
| فرایندهای در metadata | |
| فرایندهای با نواقص | |

### ۲. فهرست فرایندها (از INDEX.json)
برای هر فرایند: کد، نام فارسی، وضعیت، نواقص

### ۳. نواقص (از GAPS.json)
- فرایندهای فرعی تعریف‌نشده
- پیاده‌سازی‌های ناقص

### ۴. وابستگی‌ها
نمودار یا لیست وابستگی فرایندها به یکدیگر
